================================================================
          SpriteStudio5(sspj) Direct3D再生用サンプル
                                                  2014/07/24作成
================================================================

SpriteStudio5のプロジェクトファイル(sspj形式)を読み込み、
Direct3Dで再生するためのサンプルプログラムです。

ビルドには「Visual Studio2008以降」と「DirectX9.0c SDK」が必要となります。

急にこしらえたプログラムなので一部変な動作や、
見づらいコードが含まれているかもしれませんがご了承ください^^;

インスタンス機能や立体回転をサポートしています。
カラーブレンドや頂点変形、およびUVアニメーションは未実装です。

本サンプルの改造や流用(Helper部分含む)はご自由に行って頂いて構いません。
sspjファイルの読み込み部分はSpriteStudio5SDKのをそのまま流用していますので、
そちらのライセンスに従ってください。

本サンプルは基本的に無保証です。
ご利用は各自の責任でお願い致します。

また、バグを見つけた時などは下記の連絡先に報告いただければ
可能な限り対応したいと思います。
